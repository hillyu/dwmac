= dwmac-swap(1)
include::util/man-attributes.adoc[]
:manname: dwmac-swap
// tag::purpose[]
:manpurpose: Swaps the focused window with another window.
// end::purpose[]

// =========================================================== Synopsis
== Synopsis
[verse]
// tag::synopsis[]
dwmac swap [-h|--help] [--window-id <window-id>] [--swap-focus]
               [--wrap-around]
               (next|prev)

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

The operation is equivalent to dragging a window with the mouse.

// =========================================================== Options
include::util/conditional-options-header.adoc[]

-h, --help:: Print help

--window-id <window-id>::
include::./util/window-id-flag-desc.adoc[]

--swap-focus::
Swap focus away from the currently focused window.
By default, this command does not change the focused window.

--wrap-around::
Wrap around if the window is at the edge of the workspace window list.

// =========================================================== Arguments
include::./util/conditional-arguments-header.adoc[]

(next|prev)::
Swaps the focused window with the next or previous window in the order (top-to-bottom and left-to-right) of windows in the current workspace.
