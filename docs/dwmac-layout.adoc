== Synopsis
[verse]
// tag::synopsis[]
dwmac layout [-h|--help] [--window-id <window-id>]
                 (h_master-stack|v_master-stack|master-stack|horizontal|vertical|tiling|floating)...

// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

If several arguments are supplied then finds the first argument that doesn't describe the currently active layout, and applies the layout.

* Change both tiling layout and orientation in one go: `h_master-stack|v_master-stack`
* Change tiling layout but preserve orientation: `master-stack`
* Change orientation but preserve layout: `horizontal|vertical`
* Toggle floating/tiling mode: `tiling|floating`
